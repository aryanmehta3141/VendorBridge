# VendorBridge — Frontend

React 19 + TypeScript + Vite frontend for the VendorBridge procurement ERP.

See the root [`README.md`](../README.md) for full setup, workflow, and API documentation.

## Quick start

```bash
cp .env.example .env
# Set VITE_API_BASE_URL=http://localhost:3001

npm install
npm run dev
```

## Scripts

```bash
npm run dev      # Vite dev server → http://localhost:5173
npm run build    # TypeScript check + production build
npm run preview  # Preview production build
npm run lint     # ESLint
```
